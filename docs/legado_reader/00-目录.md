# 第0章 目录

> 本文档详细记录 Legado（阅读）Android 应用中文本阅读器的完整实现，供 Flutter 重构参考。

## 文档结构

| 文件 | 内容 |
|------|------|
| `01-架构总览.md` | 整体架构分层、核心组件关系、数据流向 |
| `02-数据模型.md` | TextChapter、TextPage、TextLine、Column 等数据结构定义 |
| `03-状态管理.md` | ReadBook 单例、三页缓冲、内容加载管线、进度持久化 |
| `04-视图层.md` | ReadView、PageView、ContentTextView 的职责与实现 |
| `05-排版引擎.md` | ChapterProvider、TextChapterLayout、ZhLayout 的排版机制 |
| `06-翻页动画.md` | PageDelegate 体系、五种翻页模式、自动翻页 |
| `07-配置系统.md` | ReadBookConfig 完整配置项、三态切换、样式更新机制 |
| `08-渲染管线.md` | 从章节内容到屏幕像素的完整渲染流程 |
| `09-Flutter重构指南.md` | Flutter 技术选型、模块划分、关键实现建议 |
