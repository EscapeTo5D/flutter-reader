# 第7章 配置系统

## 7.1 ReadBookConfig — 配置管理单例

全局单例，管理所有阅读排版配置。支持多套样式、三态切换（日/夜/EInk）。

### 代理到 Config 的配置属性

```kotlin
val config get() = if (shareLayout) shareConfig else durConfig

// 背景
var bgAlpha: Int              // 背景透明度 (0-100)

// 翻页动画
var pageAnim: Int             // 翻页动画类型 (0-4)

// 字体
var textFont: String          // 字体文件路径
var textBold: Int             // 字体粗细 (0:正常, 1:粗体, 2:细体)
var textSize: Int             // 文字大小 (sp)
var letterSpacing: Float      // 字间距

// 排版
var lineSpacingExtra: Int     // 行间距
var paragraphSpacing: Int     // 段落间距
var titleMode: Int            // 标题模式 (0:居左, 1:居中, 2:隐藏)
var paragraphIndent: String   // 段落缩进字符串

// 内边距
var paddingTop: Int           // 内容区顶部内边距
var paddingBottom: Int
var paddingLeft: Int
var paddingRight: Int
```

## 7.2 Config — 样式配置数据类

```kotlin
data class Config(
    var name: String = "",

    // 背景(三态)

    var bgStrNight: String = "#000000",        // 夜间
    var bgStrEInk: String = "#FFFFFF",         // EInk
    var bgAlpha: Int = 100,

    // 文字颜色(三态)
    private var textColor: String = "#3E3D3B",
    private var textColorNight: String = "#ADADAD",
    private var textColorEInk: String = "#000000",

    // 字体
    var textFont: String = "",
    var textBold: Int = 0,                     // 0:正常, 1:粗体, 2:细体

    var lineSpacingExtra: Int = 12,
    var paragraphSpacing: Int = 2,

    var paragraphIndent: String = "　　",       // 两个全角空格

    // 内边距 (dp)
    var paddingTop: Int = 6,
    var paddingBottom: Int = 6,
    var paddingLeft: Int = 16,
    var paddingRight: Int = 16,

    private var pageAnim: Int = 0,
    private var pageAnimEInk: Int = 4,

    // 提示信息配置

```

## 7.3 三态切换机制

| 状态 | 触发条件 | 对应字段后缀 |
|------|----------|-------------|
| 白天模式 | 默认 | 无后缀 |
| 夜间模式 | isNightTheme | `Night` |
| EInk模式 | isEInkMode | `EInk` |

```kotlin
fun curTextColor(): Int {
    return when {
        isEInkMode -> textColorEInk.toColorInt()
        isNightTheme -> textColorNight.toColorInt()
        else -> textColor.toColorInt()
    }
}

fun curPageAnim(): Int {
    return if (isEInkMode) pageAnimEInk else pageAnim
}
```

## 7.4 样式更新机制

配置变更后通过 `EventBus.UP_CONFIG` 事件通知 UI 更新：

| 级别 | 含义 | 触发的更新 |
|------|------|-----------|
| 0 | 系统UI可见性 | 更新状态栏/导航栏 |
| 1 | 背景 | 更新背景图片/颜色 |
| 8 | ChapterProvider.upStyle() | 更新画笔和排版参数 |
| 10 | 重新排版 | TextChapterLayout 重新执行 |
| 11 | 提交渲染任务 | preRenderPage() |
