## 3.2 内容加载管线

```kotlin
// 加载当前章和前后一章
fun loadContent(resetPageOffset: Boolean, success: (() -> Unit)? = null) {
    loadContent(durChapterIndex, resetPageOffset = resetPageOffset) { success?.invoke() }
    loadContent(durChapterIndex + 1, resetPageOffset = resetPageOffset)
    loadContent(durChapterIndex - 1, resetPageOffset = resetPageOffset)
}
```

contentLoadFinish 处理流程：
```
原始内容
  → ContentProcessor.getContent()   // 替换规则、标题处理
  → ChapterProvider.getTextChapterAsync()
  → 根据 offset 分发到缓冲区:
      offset == 0  → curTextChapter
      offset == -1 → prevTextChapter
      offset == 1  → nextTextChapter
  → 监听 layoutChannel, 逐页通知 UI
```

## 3.3 翻页导航

```kotlin
fun moveToNextChapter(upContent: Boolean): Boolean {
    if (durChapterIndex < simulatedChapterSize - 1) {
        durChapterPos = 0
        durChapterIndex++
        // 旋转缓冲区
        prevTextChapter = curTextChapter
        curTextChapter = nextTextChapter
        nextTextChapter = null
        loadContent(durChapterIndex + 1)
        saveRead()
        return true
    }
    return false
}

fun moveToPrevChapter(upContent: Boolean, toLast: Boolean = true): Boolean {
    if (durChapterIndex > 0) {
        durChapterPos = if (toLast) prevTextChapter?.lastReadLength ?: Int.MAX_VALUE else 0
        durChapterIndex--
        // 旋转缓冲区
        nextTextChapter = curTextChapter
        curTextChapter = prevTextChapter
        prevTextChapter = null
        loadContent(durChapterIndex - 1)
        return true
    }
    return false
}
```

## 3.4 进度持久化

```kotlin
fun saveRead(pageChanged: Boolean = false) {
    book?.let { book ->
        book.durChapterIndex = durChapterIndex
        book.durChapterPos = durChapterPos
        book.durChapterTime = System.currentTimeMillis()
        Coroutine.async { appDb.bookDao.update(book) }
    }
}
```

## 3.5 ContentProcessor — 内容处理器

```kotlin
class ContentProcessor private constructor(bookName: String, bookOrigin: String) {
    companion object {
        fun get(book: Book): ContentProcessor  // 单例缓存(WeakReference)
    }

    fun getContent(book, chapter, content, includeTitle, useReplace, chineseConvert, reSegment): BookContent
}
```

处理流程：
```
原始内容
  → 去除重复标题
  → 重新分段 (reSegment)
  → 简繁转换 (chineseConvert)
  → HTML特殊格式占位处理
  → 替换净化规则 (useReplace)
  → 还原HTML占位
  → 重新添加标题 (includeTitle)
  → 按 \n 分割并添加段落缩进
  → 返回 BookContent
```
