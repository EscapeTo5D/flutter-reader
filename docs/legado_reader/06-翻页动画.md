# 第6章 翻页动画系统

## 6.1 翻页动画类继承体系

```
PageDelegate (abstract)
│   ├── HorizontalPageDelegate (abstract) — 水平翻页基类
│   │     ├── CoverPageDelegate           — 覆盖翻页
│   │     ├── SlidePageDelegate           — 滑动翻页
│   │     ├── SimulationPageDelegate      — 仿真翻页(贝塞尔曲线)
│   │     └── NoAnimPageDelegate          — 无动画
│   └── ScrollPageDelegate                — 滚动翻页
```

## 6.2 PageDelegate — 翻页动画基类

### 字段

```kotlin
abstract class PageDelegate(protected val readView: ReadView) {
    protected val nextPage: PageView get() = readView.nextPage
    val curPage: PageView get() = readView.curPage
    protected val prevPage: PageView get() = readView.prevPage

    protected val scroller: Scroller by lazy { Scroller(context, LinearInterpolator()) }

    var isMoved = false        // 是否已移动
    var mDirection = PageDirection.NONE
    var isCancel = false       // 是否取消
    var isRunning = false      // 动画是否运行中
}
```

### 抽象方法

```kotlin
abstract fun abortAnim()
abstract fun onAnimStart(animationSpeed: Int)
abstract fun onDraw(canvas: Canvas)
abstract fun onAnimStop()
abstract fun nextPageByAnim(animationSpeed: Int)
abstract fun prevPageByAnim(animationSpeed: Int)
abstract fun onTouch(event: MotionEvent)
```

## 6.3 HorizontalPageDelegate — 水平翻页基类

```kotlin
abstract class HorizontalPageDelegate(readView: ReadView) : PageDelegate(readView) {
    protected var curRecorder = CanvasRecorderFactory.create()
    protected var prevRecorder = CanvasRecorderFactory.create()
    protected var nextRecorder = CanvasRecorderFactory.create()

    // 截图准备
    open fun setBitmap() {
        when (mDirection) {
            PREV -> { prevPage.screenshot(prevRecorder); curPage.screenshot(curRecorder) }
            NEXT -> { nextPage.screenshot(nextRecorder); curPage.screenshot(curRecorder) }
        }
    }

    // 触摸事件分发
    override fun onTouch(event: MotionEvent) {
        when (event.action) {
            ACTION_DOWN -> abortAnim()
            ACTION_MOVE -> onScroll(event)
            ACTION_UP -> onAnimStart(readView.defaultAnimationSpeed)
        }
    }

    // 滑动方向判断
    private fun onScroll(event: MotionEvent) {
        if (!isMoved) {
            if (sumX - startX > 0) setDirection(PREV)   // 向右滑 → 上一页
            else setDirection(NEXT)                      // 向左滑 → 下一页
        }
    }
}
```

## 6.4 五种翻页动画

### SimulationPageDelegate — 仿真翻页(贝塞尔曲线)

最复杂的动画，使用贝塞尔曲线模拟真实纸张翻动。

#### 核心字段

```kotlin
// 两条贝塞尔曲线的控制点
private val mBezierStart1 = PointF()    // 起始点1
private val mBezierControl1 = PointF()  // 控制点1
private val mBezierVertex1 = PointF()   // 顶点1
private var mBezierEnd1 = PointF()      // 结束点1
private val mBezierStart2 = PointF()    // 起始点2
private val mBezierControl2 = PointF()  // 控制点2
private val mBezierVertex2 = PointF()   // 顶点2
private var mBezierEnd2 = PointF()      // 结束点2

// 拖拽点对应的页脚
private var mCornerX = 1
private var mCornerY = 1
```

#### calcPoints — 贝塞尔曲线核心计算

```kotlin
private fun calcPoints() {
    mMiddleX = (mTouchX + mCornerX) / 2
    mMiddleY = (mTouchY + mCornerY) / 2

    // 控制点1: 水平方向
    mBezierControl1.x = mMiddleX - (mCornerY - mMiddleY)² / (mCornerX - mMiddleX)
    mBezierControl1.y = mCornerY

    // 控制点2: 垂直方向
    mBezierControl2.x = mCornerX
    mBezierControl2.y = mMiddleY - (mCornerX - mMiddleX)² / (mCornerY - mMiddleY)

    // 结束点 = 两直线交点
    mBezierEnd1 = getCross(touchPoint, mBezierControl1, mBezierStart1, mBezierStart2)

    // 顶点 = 二次贝塞尔中点
    mBezierVertex1.x = (mBezierStart1.x + 2*mBezierControl1.x + mBezierEnd1.x) / 4
}
```

#### onDraw — 绘制流程

```kotlin
override fun onDraw(canvas: Canvas) {
    calcPoints()
    drawCurrentPageArea(canvas, curBitmap)       // 当前页(裁剪翻页区域)
    drawNextPageAreaAndShadow(canvas, nextBitmap) // 下一页+背面阴影
    drawCurrentPageShadow(canvas)                 // 翻起页阴影
    drawCurrentBackArea(canvas, curBitmap)        // 翻起页背面(矩阵变换)
}
```

### ScrollPageDelegate — 滚动翻页

```kotlin
class ScrollPageDelegate(readView: ReadView) : PageDelegate(readView) {
    private val mVelocity: VelocityTracker = VelocityTracker.obtain()

    override fun onScroll() {
        curPage.scroll((touchY - lastY).toInt())  // 直接滚动
    }

    override fun onAnimStart(animationSpeed: Int) {
        fling(0, touchY, 0, mVelocity.yVelocity, ...)  // 惯性滚动
    }

    // 计算翻页偏移(保留一行的翻页偏移)
    private fun calcNextPageOffset(): Int {
        val lastLineTop = visiblePage.lines.last().lineTop.toInt()
        return -(lastLineTop - ChapterProvider.paddingTop)
    }
}
```

## 6.5 AutoPager — 自动翻页

### 两种模式

| 模式 | 实现方式 |
|------|----------|
| 普通模式 | 基于时间差计算滚动偏移，绘制分割线从上到下移动 |
| 墨水屏模式 | postDelayed 定时翻整页 |

### computeOffset — 计算滚动偏移

```kotlin
fun computeOffset() {
    val elapsedTime = currentTime - lastTimeMillis
    val readTime = ReadBookConfig.autoReadSpeed * 1000.0
    scrollOffsetRemain += height / readTime * elapsedTime

    if (!readView.isScroll) {
        progress += scrollOffset
        if (progress >= height) {
            if (!readView.fillPage(NEXT)) stop() else reset()
        }
    }
}
```
