## 8.2 渲染优化机制

### CanvasRecorder — Canvas录制回放

将页面内容录制到离屏 Canvas，后续重绘时直接回放，避免重复绘制计算。

```
首次绘制:
  CanvasRecorder.record(canvas) {
    // 遍历所有 TextLine, 逐个绘制
  }

后续重绘:
  CanvasRecorder.draw(canvas) {
    // 直接回放录制的内容
  }
```

### 预渲染线程

```kotlin
companion object {
    private val renderThread = Executors.newSingleThreadExecutor()
}

private fun preRenderPage() {
    renderThread.execute {
        pageFactory.run {
            if (hasPrev()) prevPage.render(view)
            curPage.render(view)
            if (hasNext()) nextPage.render(view)
        }
        postInvalidate()
    }
}
```

### 缓存失效

```kotlin
// TextPage: 使所有行缓存失效
fun invalidateAll() {
    for (line in textLines) line.invalidate()
    canvasRecorder.invalidate()
}
```

## 8.3 文字绘制详细流程

### TextPage.draw

```kotlin
fun draw(view, canvas, relativeOffset: Float) {
    // 优化模式: 使用录制回放
    if (canvasRecorder.hasRecorded()) {
        canvas.translate(0f, relativeOffset)
        canvasRecorder.draw(canvas)
        return
    }
    // 普通模式: 逐行绘制
    for (line in textLines) {
        if (!line.isVisible(relativeOffset)) continue
        canvas.translate(0f, relativeOffset)
        line.draw(view, canvas)
    }
}
```

### TextColumn.draw

```kotlin
override fun draw(view, canvas) {
    // 绘制选中背景
    if (selected) canvas.drawRect(start, lineTop, end, lineBottom, selectedPaint)
    // 绘制搜索结果高亮
    if (isSearchResult) canvas.drawRect(start, lineTop, end, lineBottom, searchPaint)
    // 绘制文字
    canvas.drawText(charData, start, lineBase, contentPaint)
}
```

### ImageColumn.draw

```kotlin
override fun draw(view, canvas) {
    val bitmap = ImageProvider.getImage(src)
    // 计算缩放比例, 居中绘制
    val scale = min(width / bitmap.width, height / bitmap.height)
    canvas.drawBitmap(bitmap, srcRect, dstRect, imagePaint)
}
```
